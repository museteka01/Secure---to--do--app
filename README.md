# Secure---to--do--app
demo dev sec ops project:  python app with CI/CD pipeline (tests, SAST, secrets, dependency and IaC scans 

This project demonstrates  alight weight devsec ops  work flow using a simple  python Flask app 

It includes automated unit tests, security  scans (Bnadit, semgrep, Gitleaks, Pip-audit) and infrastructure-as-acode validation  with Terraform  and tfsec, all running in the git hub action CI
